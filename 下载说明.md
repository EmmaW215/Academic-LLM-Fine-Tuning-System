# 📦 代码压缩包下载说明

## 📥 下载文件

压缩包文件名格式：`Academic-LLM-Fine-Tuning-System-YYYYMMDD-HHMMSS.tar.gz`

## 📋 压缩包内容

### ✅ 包含的文件
- ✅ 所有源代码（Python 文件）
- ✅ 配置文件（`config/` 目录）
- ✅ 所有模块（`modules/` 目录）
- ✅ 文档文件（README, 指南等）
- ✅ 依赖列表（`requirements-file.txt`）
- ✅ 脚本文件（`.sh` 脚本）

### ❌ 排除的文件（大文件/敏感信息）
- ❌ 模型文件（`storage/models/` - 约 2.2GB）
- ❌ 数据文件（`storage/data/` - PDF、JSON 等）
- ❌ 索引文件（`storage/indexes/` - FAISS、SQLite）
- ❌ 环境变量文件（`.env` - 包含 API keys）
- ❌ 缓存文件（`__pycache__`, `.gradio`）
- ❌ 虚拟环境（`.venv`, `venv`）
- ❌ Git 历史（`.git`）

## 🚀 本地使用步骤

### 1. 解压文件
```bash
tar -xzf Academic-LLM-Fine-Tuning-System-*.tar.gz
cd Academic-LLM-Fine-Tuning-System-*
```

### 2. 安装依赖
```bash
pip install -r requirements-file.txt
```

### 3. 配置环境变量
```bash
# 创建 .env 文件
cp .env.example .env  # 如果有的话
# 或手动创建
echo "OPENAI_API_KEY=your_key_here" > .env
```

### 4. 下载模型（如果需要）
模型会从 HuggingFace 自动下载到 `~/.cache/huggingface/`

### 5. 运行项目
```bash
# Gradio UI
python gradio-ui.py

# 或 FastAPI
uvicorn module8-api:app --host 0.0.0.0 --port 8000
```

## 📊 文件大小

压缩包大小通常在 **5-20 MB** 左右（取决于代码量），不包含模型和数据文件。

## ⚠️ 注意事项

1. **模型文件需要单独下载**：
   - 基础模型会从 HuggingFace 自动下载
   - 微调后的适配器需要从 GPU 服务器单独下载（如果需要）

2. **数据文件需要重新生成**：
   - 运行数据收集步骤重新下载论文
   - 或从 GPU 服务器单独下载数据文件

3. **索引需要重建**：
   - 运行 RAG 索引步骤重新构建索引
   - 或从 GPU 服务器单独下载索引文件

## 🔄 更新压缩包

如果需要更新压缩包，运行：
```bash
./create_download_archive.sh
```
