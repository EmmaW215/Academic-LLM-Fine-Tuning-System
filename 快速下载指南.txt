========================================
快速下载项目指南
========================================

【方法 1: JupyterLab 文件浏览器（最简单）】

1. 在 JupyterLab 左侧文件浏览器中
2. 找到 /home/jovyan/work 文件夹
3. 右键点击 → 选择 "Download"
4. 浏览器会自动下载 ZIP 文件

【方法 2: 下载代码压缩包（推荐）】

已为您创建了代码压缩包（排除大数据）：
  /home/jovyan/academic-llm-code.tar.gz

在 JupyterLab 文件浏览器中：
1. 导航到 /home/jovyan/
2. 找到 academic-llm-code.tar.gz
3. 右键 → Download

【方法 3: 推送到 GitHub】

运行以下命令：

1. 初始化 Git:
   cd /home/jovyan/work
   git init
   git add .
   git commit -m "Initial commit"

2. 在 GitHub 创建新仓库

3. 推送:
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main

详细说明请查看: DOWNLOAD_AND_PUSH_GUIDE.md
